Flex

  • To Use flexbox

  • Css properties for flexbox
    • 1:flex-direction: column, row, column-reverse, row-reverse
    2:flex-wrap: nowrap, wrap, wrap-reverse
    3:flex-flow: shorthand for both flexi-direction and flex-wrap :: flex-flow: row wrap
    4:justify-content: center, flex-start, flex-end, space-around, space-between, space-evenly
    5:align-items: center, flex-start, flex-end, stretch, baseline, normal
    6:align-content: center, stretch, flex-start, flex-end, space-around, space-between, space-evenly,
    7:flex-items: order, flex-grow, flex-shrink, flex-basis, flex, align-self
